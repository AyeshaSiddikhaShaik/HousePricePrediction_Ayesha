# House Price Prediction Using Machine Learning

## An Explainable AI Approach for Smart Real Estate Valuation

### Project Overview

This project focuses on predicting house prices using Machine Learning techniques and real-world housing data. The objective is to develop an accurate and interpretable predictive model that estimates property prices based on various housing features such as area, bedrooms, bathrooms, parking facilities, furnishing status, and other amenities.

The project follows a complete Data Science workflow, including data exploration, data cleaning, feature engineering, exploratory data analysis (EDA), model development, model evaluation, and explainable AI-based insights.

### Dataset

Dataset: Housing Prices Dataset

Source: Kaggle

The dataset contains information about residential properties and their corresponding market prices.

### Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation
7. Visualization
8. Feature Importance Analysis
9. Explainable AI Insights
10. Business Recommendations

### Machine Learning Models Used

* Linear Regression
* Random Forest Regressor

### Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

### Key Findings

* Bathrooms showed the strongest influence on house prices.
* Air conditioning and hot water heating positively impacted property valuation.
* Preferred area and number of stories contributed significantly to price prediction.
* Linear Regression achieved the best overall performance with an R² Score of approximately 65%.
